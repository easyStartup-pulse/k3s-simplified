<h2 align="center">k3s-simplified</h2>

<p align="center">
  <strong>k3s-simplified</strong> is a streamlined CLI tool for setting up and managing production-grade Kubernetes clusters on <a href="https://hetzner.com" target="_blank">Hetzner Cloud</a> using K3s provided by Rancher. Compatible with <strong>Linux</strong>, <strong>macOS</strong>, and <strong>Windows (via WSL)</strong>, it also facilitates private network Kubernetes nodes without public IPs, enhancing security.
</p>
